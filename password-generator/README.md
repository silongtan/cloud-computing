# Password Generator

A rust based password generator that supports customize length and complexity.

## Module Used

Random number generator: [rand](https://crates.io/crates/rand)

## Usage

### Clone the repository

```git clone [\this repo\]```

### Build the project

```cargo build```

### Run the project

```cargo run -- --length [LENGTH] --simple [OPTIONAL]```
